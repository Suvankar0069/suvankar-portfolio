Diabetes Prediction Using Machine Learning

Project Summary

Diabetes Prediction is a machine learning application that predicts whether a person is likely to have diabetes based on medical parameters such as glucose level, blood pressure, BMI, insulin level, age, and other health-related features. The model was trained using the Pima Indians Diabetes Dataset to provide fast and reliable predictions.

Features

  • Predicts diabetes risk from user health data.
  • Clean and responsive web interface.
  • Trained using Support Vector Machine (SVM).
  • Displays instant prediction results.
  • Simple and user-friendly design.

Technologies Used

  • Python
  • Machine Learning
  • Scikit-learn
  • Pandas
  • NumPy
  • Streamlit
  • HTML
  • CSS

My Role

Developed the machine learning model, performed data preprocessing, trained the Support Vector Machine (SVM) classifier, evaluated its performance, and built a simple web interface to make predictions.
